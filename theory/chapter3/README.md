# Chapter 3

## Node.js File System Module

<br />

<h3>The Node.js file system module allows you to work with the file system on your computer.</h3>
<h5>Common use for the File System module:</h5>
<ul>
    <li>Read files</li>
    <li>Create files</li>
    <li>Update files</li>
    <li>Delete files</li>
    <li>Rename files</li>
</ul>
